## Lokal utvikling

Vil du utvikle direkte mot API-et uten en lokal `floq`-instans?

Legg tokendata (se under) i `src/common/tokenData.dev.local.json`

```json
{
  "id_token": "<>",
  "apiToken": "<>",
  "config": { "apiUri": "<TEST_URL>" },
  "userEmail": "<>"
}
```
